# netflix-data-analysis-eda
Netflix dataset analysis using EDA techniques. Performed data cleaning, feature engineering, and visualizations to extract actionable insights.
# Netflix Data Analysis Project

## Objective
Perform Exploratory Data Analysis (EDA) on Netflix dataset.

## Steps Performed
- Data Cleaning
- Handling Missing Values
- Feature Engineering
- Data Visualization

## Key Insights
- USA produces the most content
- Movies dominate over TV shows
- Content growth increased after 2015
- Netflix expanded rapidly in recent years

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
